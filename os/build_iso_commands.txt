Phoenix-15 ISO Build (Ubuntu Live USB)

1) Install build tools:
sudo apt update
sudo apt install -y live-build debootstrap xorriso syslinux grub-pc-bin grub-efi-amd64-bin mtools dosfstools squashfs-tools rsync curl ca-certificates git

2) Clone + build:
git clone https://github.com/CydonianHeavyIndustries/Phoenix-15.git
cd Phoenix-15
sudo bash os/build_iso.sh

3) Output ISO:
os/out/phoenix-15.iso
