Phoenix-15 Offline Docs

Quick Start

Primary UI launches automatically on boot. Use Phoenix Control Center for system tools.

Update From USB ZIP

Control Center → Update from USB Zip.

ZIP can contain app/ at root or the full repo. A backup is created before applying.

Game Server

Control Center → Game Server for SteamCMD profiles and installs.

Network Safe Mode

Control Center → Network Safe Mode to disable networking until you re-enable it.

Integrity Check

Control Center → Integrity Check to verify Phoenix files against the latest manifest.
