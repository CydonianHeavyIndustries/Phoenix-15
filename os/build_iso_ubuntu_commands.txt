Ubuntu ISO build (Phoenix-15)

1) Clone/update a low-footprint workspace (recommended)
rm -rf ~/Phoenix-15
git clone --depth 1 --filter=blob:none --sparse https://github.com/CydonianHeavyIndustries/Phoenix-15.git ~/Phoenix-15
cd ~/Phoenix-15
git sparse-checkout set os
bash os/bootstrap_iso_workspace.sh --force

2) Run the Ubuntu build helper
bash ~/Phoenix-15/os/build_iso_ubuntu.sh

3) ISO output (if successful)
~/Phoenix-15/os/out/phoenix-15.iso

4) Logs
~/Phoenix-15/os/out/build-iso.log
~/Phoenix-15/os/out/build-iso-ubuntu-YYYYMMDDTHHMMSSZ.log
